              <div class='synopsis'>
                <span class='name'>descendents</span>
                <span class='arguments'>(*args)</span>
              </div>
              <div class='description'>
                <p>
                Returns all descendents of the current node. Each level is within its own
                hash, so for a structure like:
                </p>
                <pre>root&#x000A; \_ child1&#x000A;      \_ subchild1&#x000A;            \_ subsubchild1&#x000A;      \_ subchild2</pre>
                <p>
                the resulting hash would look like:
                </p>
                <pre>{child1 =&gt;&#x000A;  {subchild1 =&gt;&#x000A;    {subsubchild1 =&gt; {}},&#x000A;   subchild2 =&gt; {}}}</pre>
                <p>
                This method will accept two parameters.
                </p>
                <pre>* :raw -&gt; Result is flat array. No Hash tree is built&#x000A;* {:depth =&gt; n} -&gt; Will only search for descendents to the given depth of n</pre>
              </div>
